Static (default)
The position is whatever the browser gives it, when it calculates the page flow; you can't change it later with top/left offsets.
Relative Offsets do work, and are calculated relative to the default (static) position.
Absolute
Offsets work, and are calculated relative to the nearest containing element which has a specific position value... otherwise relative to the tag.
Fixed
Offsets calculated relative to the window (viewport), not the document - i.e. does not scroll with the document.
